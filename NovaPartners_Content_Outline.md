## 2. Hero Section (시각적 임팩트 최우선)

**레이아웃**: 풀스크린, 좌측 텍스트 + 우측 비주얼

### **좌측 콘텐츠**
- **주요 카피** (한 줄 카피)
  - "AI가 만드는 신뢰할 수 있는 미래"
  - 또는 "데이터는 믿음이 되어야 한다"
  
- **서브 카피** (2-3줄)
  - 솔루션이 제공하는 핵심 가치 한 문장
  
- **CTA 버튼** (2개)
  - Primary: "데모 신청하기" → `/solutions/ai-platform/demo`
  - Secondary: "솔루션 보기" → `/solutions/ai-platform/`

### **우측 비주얼** 🎬
- **배경 동영상** (무음, 자동 재생, 루프)
  - 주제: AI, 데이터 분석, 신뢰성 (약 5-8초)
  - 형식: MP4 또는 WebM (최적화)
  
- **또는 애니메이션 일러스트레이션**
  - 마우스 움직임에 따른 **패럴렉스 효과** (3D 깊이감)
  - 스크롤 시 애니메이션 트리거

### **시각적 효과**
- ✨ Hero 배경에 **그래디언트 오버레이** (어두운 톤)
- ✨ 텍스트는 흰색, 명확한 대비
- ✨ 버튼 Hover 시 **배경색 변화 + 그림자 강화**

3. **의료** (`/solutions/industries/healthcare`)
   - 배경 이미지: 의료 데이터/의사와 환자

4. **소매** (`/solutions/industries/retail`)

- 🎬 텍스트 위에 **반투명 오버레이** (텍스트 가독성 확보)
- 🎬 통계 숫자는 **색상 강조** (브랜드 색상)

- **Carousel**: 자동 5초, Manual 즉시

### **이미지 & 비디오 최적화**
- 모든 이미지: WebP 형식 (fallback PNG)
- 비디오: MP4 + WebM (모바일/데스크톱)
- 자동재생 비디오: 무음 (muted="true")

### **마우스 인터렉션 요약**
1. **Hover 효과**: 배경색 변화, 그림자 상승, 스케일 증가
2. **클릭**: 페이지 이동 또는 모달 오픈
3. **스크롤 트리거**: 애니메이션 진행 (카운팅, 차트 등)
4. **드래그**: 슬라이더, 캐러셀, Before/After 이미지
5. **포커스**: 입력 필드 테두리 변화
