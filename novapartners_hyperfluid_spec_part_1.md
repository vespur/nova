# 2. URL STRUCTURE (2026 SPATIAL INTERPRETATION)

## 2.1 URL = Spatial Layer Path
URL은 단순한 주소가 아니라 **Layer 간 이동 규칙**으로 재해석된다.

예시:
- `/solutions/ai-platform/` = Tech Assembly Scene
- `/company/history` = Timeline Fold Scene
- `/support/faq` = Utility Knowledge Scene

## 2.2 3대 Macro Zone
- **Brand Zone:** 빛의 방향 변화(Plasma Beam)
- **Solutions Zone:** 구조적 조립/해체(Assembly Motion)
- **Support Zone:** Glass-panel 기반 안정적 전환

## 2.3 URL Depth → Motion Depth
- Depth 1: Hero 중심 (Z 60~100)
- Depth 2: 정보형 콘텐츠 (Z 20~40)
- Depth 3: 백그라운드/Utility (Z 0~15)
